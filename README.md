# 🌌 The Magic Rift

**The Magic Rift** is a 2D action-shooter built with Python and the **Pygame** library. 

This project was inspired by the "Alien Invasion" game from the book *Python Crash Course* by Eric Matthes. However, I have significantly customized the game to make it my own, including a complete visual overhaul and enhanced gameplay mechanics.

---

### 🛠 Key Features & Enhancements:
* **Full 4-Way Movement:** Unlike the original version, I implemented vertical movement (Up/Down) alongside horizontal movement, making the gameplay more dynamic.
* **Custom Theme:** Reimagined the visual assets and atmosphere to fit the "Magic Rift" theme.
* **Object-Oriented Design:** The project is structured using clean OOP principles, with separate classes for the mage (player), projectiles, enemies, and game settings.
* **Dynamic Difficulty:** The game scales in difficulty as you progress.

---

### 🎮 Controls:
* **Arrow Keys:** Move the character (Up, Down, Left, Right).
* **Spacebar:** Shoot magic orbs.
* **Esc:** Exit the game.
* **P:** Start the game (from the main menu).

---

### 🚀 Getting Started:

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/zllrocky/the-magic-rift.git](https://github.com/zllrocky/the-magic-rift.git)
    ```
2.  **Install dependencies:**
    Make sure you have Python installed, then run:
    ```bash
    pip install -r requirements.txt
    ```
3.  **Run the game:**
    ```bash
    python mage_invasion.py
    ```
    *(Note: Replace 'mage_invasion.py' with the name of your main file, e.g., 'alien_invasion.py' or 'main.py')*
